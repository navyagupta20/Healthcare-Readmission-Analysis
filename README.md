# 📊 Healthcare Readmission Analysis

---

## 🏥 **Project Overview**

This project analyzes a large-scale diabetes hospital dataset (1999–2008) from 130 US hospitals to understand patient behavior, hospital utilization patterns, and factors influencing **30-day hospital readmission risk**.

The goal is to extract actionable insights that can help healthcare providers reduce readmission rates and improve patient care efficiency.

---

## 🎯 **Objectives**

- Analyze patient demographics and medical history  
- Study hospital utilization patterns (stay duration, medications, diagnoses)  
- Identify key factors affecting readmission risk  
- Compute and visualize 30-day readmission rate  
- Build an interactive dashboard for insights  
- Generate business/healthcare recommendations  

---

## 📂 **Dataset Information**

- **Source:** Diabetes 130-US hospitals dataset (1999–2008)  
- **Records:** 101,766  
- **Features:** 49  
- **Type:** Real-world clinical dataset  

---

## 🧹 **Data Preprocessing**

- Handled missing values  
- Removed irrelevant or noisy columns  
- Standardized categorical variables  
- Converted age groups into numerical format  
- Cleaned readmission labels (`<30`, `>30`, `NO`)  
- Feature engineering for analysis readiness  

---

## 📊 **Exploratory Data Analysis (EDA)**

### Key analysis performed:

- Patient age distribution  
- Gender distribution  
- Hospital stay duration analysis  
- Medication usage patterns  
- Diagnosis category trends  
- Readmission behavior across groups  

---

## 📈 **Key Insights**

- Average patient age: ~66 years  
- Average hospital stay: ~4.4 days  
- Average medications per patient: ~16  
- 30-day readmission rate: ~11.16%  

### **Important Findings:**

- Patients with longer hospital stays showed different readmission patterns  
- Medication count is linked with patient severity  
- Certain diagnosis categories dominate admissions  
- Age group significantly affects hospitalization trends
